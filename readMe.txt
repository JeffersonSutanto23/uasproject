Aplikasi ini dapat dijalankan dengan beberapa langkah :
- Aktifkan XAMPP pada bagian Apache dan MySQL
- untuk mengakses bagian pelanggan, ketik di browser website web1.desdulianto.com/group05/(nama file)
- untuk mengakses bagian admin, ketik di browser website web1.desdulianto.com/group05/administrator/(nama file)
- sebelum akses ke bagian index, pengguna melakukan login terlebih dahulu dan jika pengguna merupakan pelanggan maka pengguna bisa melakukan register
